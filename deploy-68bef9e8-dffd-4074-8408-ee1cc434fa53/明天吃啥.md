# 明天吃啥

> 日期：2026-06-10  
> 目标：吃得舒服、简单、不太费脑子。

## 早餐

**豆浆 + 鸡蛋 + 全麦面包**

理由：清淡、顶饱，早上不会太腻。

```text
早餐搭配：
- 无糖豆浆 1 杯
- 水煮蛋 1-2 个
- 全麦面包 1-2 片
```

## 午餐

**番茄牛肉饭**

理由：有肉、有菜、有主食，味道也比较稳。

```yaml
午餐:
  主食: 米饭
  蛋白质: 牛肉
  蔬菜: 番茄
  口味: 咸香微酸
```

## 晚餐

**清炒时蔬 + 鸡胸肉/鱼肉 + 紫菜蛋花汤**

理由：晚上吃轻一点，睡前负担小。

```json
{
  "晚餐": ["清炒时蔬", "鸡胸肉或鱼肉", "紫菜蛋花汤"],
  "建议": "少油少盐，别吃太撑"
}
```

## 小零食

如果下午饿了，可以吃：

- 香蕉
- 酸奶
- 坚果一小把
